CODE FESTIVAL 2016 Qualification Round A

еҸ‘иЎЁдәҺ 2016-09-27   |   еҲҶзұ»дәҺ йўҳи®°   |  

йўҳзӣ®й“ҫжҺҘдёҺд»Јз Ғй“ҫжҺҘ

иҝҳжҳҜйӮЈеҸҘиҜқеҗ§пјҢеҘҪеҘҪжҖқиҖғпјҢе№¶дё”еёҢжңӣиҮӘе·ұиғҪеқҡжҢҒдёӢеҺ»гҖӮ

A - CODEFESTIVAL 2016

йўҳж„ҸжҳҜз»ҷдёҖдёӘй•ҝеәҰдёә12зҡ„еӯ—з¬ҰдёІпјҢиҫ“еҮәеүҚ4дёӘеӯ—з¬Ұ+з©әж ј+еҗҺеӣӣдёӘеӯ—з¬ҰгҖӮ

B - д»ІиүҜгҒ—гҒҶгҒ•гҒҺ / Friendly Rabbits

йўҳж„ҸжҳҜз»ҷеҮәNдёӘе…”еӯҗпјҢжҜҸдёӘе…”еӯҗжңүиҮӘе·ұе–ңж¬ўзҡ„е…”еӯҗ$A_i$пјҢй—®дә’зӣёе–ңж¬ўзҡ„е…”еӯҗжңүеӨҡе°‘еҜ№гҖӮ

зӣҙжҺҘжЁЎжӢҹи®Ўз®—гҖӮ

C - ж¬ЎгҒ®гӮўгғ«гғ•гӮЎгғҷгғғгғҲ / Next Letter

йўҳж„ҸжҳҜз»ҷеҮәдёҖдёӘеӯ—з¬ҰдёІпјҢеӯ—з¬ҰдёІдёӯзҡ„жҜҸдёҖдёӘеӯ—з¬ҰеҸҜд»ҘеҫӘзҺҜеҗ‘еҗҺз§»пјҢдҪҶжҳҜдёҖе…ұеҝ…йЎ»еҫӘзҺҜеҗ‘еҗҺз§»еҠЁKж¬ЎпјҢй—®ж”№еҸҳеҗҺзҡ„еӯ—е…ёеәҸжңҖе°Ҹзҡ„еӯ—з¬ҰдёІгҖӮ

еҜ№жҜҸдёҖдёӘеӯ—з¬ҰиҙӘеҝғжҗһпјҢеү©дёӢзҡ„ж“ҚдҪңеңЁжңҖеҗҺдёҖдёӘеӯ—з¬ҰдёҠгҖӮ

D - гғһгӮ№зӣ®гҒЁж•ҙж•° / Grid and Integers

йўҳж„ҸжҳҜз»ҷеҮәдәҶR*Cзҡ„ж јеӯҗпјҢз„¶еҗҺжңүNдёӘж јеӯҗ($r_i,c_i$)е·Із»ҸжңүеҖјдәҶ$a_i$пјҢзҺ°еңЁиҰҒжұӮи®©дҪ еҲӨж–ӯжҳҜеҗҰжҜҸдёҖдёӘж јеӯҗеңЁйғҪжҳҜеӨ§дәҺзӯүдәҺйӣ¶зҡ„жғ…еҶөдёӢпјҢж»Ўи¶іеңЁд»»ж„ҸдёҖдёӘ2*2зҡ„ж јеӯҗдёӯпјҢдёӨдёӘеҜ№и§’зәҝзҡ„е’ҢжҳҜзӣёзӯүзҡ„гҖӮ

еҸҲжҳҜиҝҷз§ҚиҰҒдҪ е°Ҷйўҳзӣ®иҪ¬еҢ–зҡ„йўҳгҖӮ

йҰ–е…ҲеҜ№дәҺд»»ж„ҸдёҖдёӘ2*2зҡ„ж–№ж јпјҢиҰҒж»Ўи¶і
$$
b_{i,j}+b_{i+1,j+1}=b_{i+1,j}+b_{i,j+1}
$$
зӯүејҸжҢӘдёҖдёӢеҸҳжҲҗе·®зҡ„еҪўејҸпјҢдјҡеҸ‘зҺ°еҜ№дәҺеӣәе®ҡзҡ„дёҖиЎҢiжқҘиҜҙпјҢе…¶е·®еҖјжҳҜиҰҒзӣёзӯүзҡ„гҖӮеҜ№дәҺеӣәе®ҡзҡ„дёҖеҲ—jжқҘиҜҙпјҢе…¶е·®еҖјд№ҹжҳҜзӣёзӯүзҡ„гҖӮйӮЈд№ҲиҝҷдёӘж—¶еҖҷжҖқиҖғпјҢеҸҜд»Ҙе°ҶдёҠиҝ°жқЎд»¶иҪ¬еҸҳдёәжҳҜеҗҰеӯҳеңЁдёӨдёӘж•°еҲ—$(x_1,x_2,вҖҰ,x_R)$е’Ң$(y_1,y_2,вҖҰ,y_C)$пјҢдҪҝеҫ—$x_{rk}+y_{ck}=a_k$гҖӮ

жүҖд»Ҙж №жҚ®йўҳдёӯжүҖз»ҷзҡ„жқЎд»¶еҸҜд»Ҙжһ„е»әеӣҫпјҢжҜҸдёҖдёӘиҫ№зҡ„жқғеҖје°ұжҳҜ$a_k$пјҢеҪ“еҪўжҲҗзҺҜзҡ„ж—¶еҖҷеҲӨж–ӯжҳҜеҗҰзӣёзӯүгҖӮ

жңҖеҗҺпјҢиҰҒеҲӨж–ӯжҳҜеҗҰйғҪж»Ўи¶іжҳҜеӨ§дәҺзӯүдәҺ0пјҢжҲ‘д»¬е…ҲеҒҮи®ҫжңҖејҖе§Ӣзҡ„еҖјxжҳҜжңҖе°ҸеҖј0пјҢж №жҚ®иҝҷдёӘеҹәеҮҶпјҢз®—е…¶дҪҷзӮ№зҡ„еҖјпјҢи®Ўз®—жңҖе°ҸжӯЈеҖјдёҺжңҖе°ҸиҙҹеҖјпјҢеҰӮжһңжңҖе°ҸиҙҹеҖјзҡ„з»қеҜ№еҖјеӨ§дәҺдәҶжңҖе°ҸжӯЈеҖјпјҢз”»еӣҫеҸҜзҹҘпјҢиҝҷдёӘж—¶еҖҷе°ұеҲ°дәҶдёҚеҸҜи°ғе’Ңзҡ„ең°жӯҘпјҢдёҚз¬ҰеҗҲиҰҒжұӮгҖӮ

E - LRU гғ‘гӮәгғ« / LRU Puzzle

йўҳж„ҸжҳҜжңүNдёӘ1еҲ°MйҖ’еўһзҡ„ж•°з»„пјҢз»ҷеҮәдәҶдёҖзі»еҲ—зҡ„ж“ҚдҪңпјҢжҜҸж¬Ўж“ҚдҪңе°Ҷд»ҺNдёӘж•°з»„дёӯйҖүеҮәдёҖдёӘж•°з»„пјҢе№¶е°ҶиҝҷдёӘж•°з»„дёӯзҡ„$a_i$жӢҝеҲ°жңҖеүҚйқўгҖӮзҺ°еңЁйўҳзӣ®жҳҜй—®жҳҜеҗҰж»Ўи¶іеңЁиҝӣиЎҢQж¬Ўж“ҚдҪңд№ӢеҗҺпјҢеҫ—еҲ°зҡ„з»“жһңжҳҜзӣёзӯүзҡ„гҖӮ

еҒҮи®ҫж“ҚдҪңеңЁ(1,2,вҖҰ,M)дёҠпјҢйӮЈд№ҲеңЁиҝӣиЎҢдәҶQж¬Ўж“ҚдҪңд№ӢеҗҺеҫ—еҲ°зҡ„з»“жһңпјҢе°ұжҳҜд»ҺеҗҺеҫҖеүҚзңӢпјҢеҪ“иҝҷдёӘж•°з¬¬дёҖж¬ЎеҮәзҺ°зҡ„ж—¶еҖҷпјҢе°ҶиҝҷдёӘж•°ж”ҫеҲ°з»“жһңзҡ„йҳҹеҲ—жң«е°ҫпјҢеҰӮжһңеҮәзҺ°иҝҮпјҢеҝҪз•ҘжҺүгҖӮд№ӢеҗҺе°ҶжІЎжңүеҮәзҺ°еңЁз»“жһңйҳҹеҲ—дёӯзҡ„ж•°еӯ—дҫқж¬ЎжҸ’е…ҘеҲ°з»“е°ҫгҖӮ

йӮЈд№Ҳй—®йўҳзҺ°еңЁзӣёеҪ“дәҺе°ҶиҝҷдёӘж“ҚдҪңзҡ„ж•°з»„aпјҢеҲҶжҲҗNдёӘеӯҗдёІпјҢиҰҒжұӮиҝҷNдёӘеӯҗдёІеңЁ1..MдёҠзҡ„ж“ҚдҪңжҳҜзӣёзӯүзҡ„з»“жһңгҖӮй—®жҳҜеҗҰеҸҜиғҪгҖӮ

йӮЈйҰ–е…ҲиҖғиҷ‘N=2зҡ„жғ…еҶөгҖӮиҙӘеҝғеҺ»еҸ–пјҢзў°еҲ°жІЎеҮәзҺ°иҝҮзҡ„пјҢз»ҷз¬¬дёҖдёӘгҖӮзў°еҲ°з¬¬дёҖдёӘеҮәзҺ°зҡ„пјҢз»ҷз¬¬дәҢдёӘгҖӮйғҪеҮәзҺ°зҡ„пјҢеҝҪз•ҘжҺүгҖӮжңҖз»ҲзңӢдёӨдёӘж•°з»„еҫ—еҲ°зҡ„жҳҜеҗҰзӣёзӯүгҖӮ

еҪ“NеӨ§дәҺ2зҡ„ж—¶еҖҷпјҢжҖқи·Ҝд№ҹжҳҜдёҖж ·зҡ„пјҢзұ»дјјдәҺжҺЁз®ұеӯҗгҖӮгҖӮгҖӮ

йҰ–е…Ҳи®°й•ҝеәҰдёә0зҡ„жңүNдёӘж•°з»„пјҢз„¶еҗҺеҫ—еҲ°дёҖдёӘж•°еӯ—еҲӨж–ӯеҪ“еүҚйқўдёҙзҡ„зҠ¶жҖҒпјҢеҫҖдёӢйқўжҺЁпјҢжңҖеҗҺзңӢз®ұеӯҗжңүжІЎжңүеү©дёӢзҡ„гҖӮгҖӮгҖӮ
